Для создания типового кода есть снипеты - создание шаблонов
Для создания компоненты есть такой снипет -
rsc (react stateless component)

Когда начинаешь писать тег компоненты (в другой компоненте), то редактор дает
подсказку (название компоненты и путь) и, выбирая ее, импорт прописывается
автоматом !

Shift + F6 = поменять названия всех одинаковых переменных
Ctrl + D = копирование строки (не копируется в буфер "ctrl + c")
